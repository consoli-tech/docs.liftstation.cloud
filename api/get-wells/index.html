Get Wells

Description#

This request will retrieve a list of the wells of your organization. It is used so that you can get the well IDs, which you will need to enter a reading. Wells are grouped by the route that they belong to.

Request#

Type / Endpoint: GET https://api.liftstation.cloud/v1/wells

Response#

Status Code: 200#

Description: Request was successful.
